# Test SvelteKit App

A demo SvelteKit application for testing DevFlow onboarding.

## Features

- User authentication
- Dashboard with data visualization
- Settings management
- Dark mode support

## Roadmap

- [ ] Add user profiles

## Getting Started

```bash
npm install
npm run dev
```

## Testing

```bash
npm test
npm run test:integration
```
